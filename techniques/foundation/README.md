# Foundation Techniques

Master the core skills for precise image generation with Nano Banana Pro.

These foundational techniques are essential building blocks. Each technique teaches a specific skill that you'll combine in intermediate and expert workflows.

---

## Technique List

### [F1: Descriptive Scene Composition](./F1-descriptive-composition.md)
**Core Principle**: Write narrative prompts over keyword lists

**What You'll Learn**:
- Transform keywords into flowing scene descriptions
- Structure prompts for maximum clarity
- Balance detail with creative freedom

**Prerequisite**: None - start here!
**Time to Master**: 30 minutes
**Use Cases**: All image generation tasks

---

### [F2: Camera Control Fundamentals](./F2-camera-control.md)
**Core Principle**: Use photography vocabulary for precise composition

**What You'll Learn**:
- Shot types (close-up, wide-angle, macro, aerial)
- Camera angles (low-angle, high-angle, Dutch angle)
- Lens selection (85mm portrait, 35mm wide, telephoto)

**Prerequisite**: F1
**Time to Master**: 45 minutes
**Use Cases**: Product photography, editorial content, portraits

---

### [F3: Lighting Basics](./F3-lighting-basics.md)
**Core Principle**: Control mood and atmosphere through lighting

**What You'll Learn**:
- Natural lighting (golden hour, overcast, harsh sunlight)
- Direction (front, back, side, rim lighting)
- Quality (soft, hard, dramatic, subtle)

**Prerequisite**: F1
**Time to Master**: 45 minutes
**Use Cases**: Product shots, portraits, atmospheric scenes

---

### [F4: Style Transfer](./F4-style-transfer.md)
**Core Principle**: Apply artistic styles consistently

**What You'll Learn**:
- Art movement styles (impressionism, minimalism, art deco)
- Visual styles (photorealistic, illustration, vintage, cinematic)
- Style vocabulary and descriptors

**Prerequisite**: F1
**Time to Master**: 1 hour
**Use Cases**: Branded content, artistic projects, concept development

---

### [F5: Aspect Ratio Mastery](./F5-aspect-ratios.md)
**Core Principle**: Choose and optimize for different formats

**What You'll Learn**:
- Understanding aspect ratios (1:1, 16:9, 9:16, 4:5, etc.)
- Platform-specific optimization
- Composition adjustments for each format

**Prerequisite**: F2
**Time to Master**: 30 minutes
**Use Cases**: Social media, presentations, print materials

---

### [F6: Text Rendering Essentials](./F6-text-rendering.md)
**Core Principle**: Generate clear, legible text in images

**What You'll Learn**:
- Text specification techniques
- Font style descriptions
- Context and placement control
- Multilingual text generation

**Prerequisite**: F1
**Time to Master**: 1 hour
**Use Cases**: Posters, logos, infographics, product labels

---

## Learning Sequence

### Recommended Path
Complete techniques in order for optimal learning:

1. **F1: Descriptive Composition** (Required foundation)
2. **F2: Camera Control** OR **F3: Lighting** (Choose based on interest)
3. **F4: Style Transfer** (Builds on F1)
4. **F5: Aspect Ratios** (Quick practical skill)
5. **F6: Text Rendering** (Advanced foundation skill)

### Fast Track (2 hours)
Essential skills for immediate productivity:
1. F1: Descriptive Composition
2. F2: Camera Control
3. F6: Text Rendering

### Complete Foundation (4-5 hours)
All foundation techniques for comprehensive mastery.

## Practice Challenges

After completing foundation techniques, test your skills:

### Challenge 1: Product Showcase
**Goal**: Create a product photo using F1, F2, and F3
```
Requirements:
- Descriptive scene (F1)
- Specific camera angle (F2)
- Controlled lighting (F3)
```

### Challenge 2: Styled Scene
**Goal**: Generate an artistic scene using F1 and F4
```
Requirements:
- Narrative description (F1)
- Specific art style (F4)
- Consistent mood
```

### Challenge 3: Social Media Asset
**Goal**: Create platform-ready image using F1, F5, and F6
```
Requirements:
- Clear composition (F1)
- Optimal aspect ratio (F5)
- Legible text overlay (F6)
```
